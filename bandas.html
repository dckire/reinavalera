bandas

"bandas" se refiere a grupos de soldados o guerreros que se unen para atacar o sitiar una ciudad.

En hebreo, la palabra utilizada es "גְּדוּד" (gedud), que significa "tropa", "compañía" o "banda". En este contexto, se refiere a una unidad militar que se une para llevar a cabo una misión o una campaña.

Miqueas 4:14, la frase "hija de bandas" se utiliza para describir a la ciudad de Jerusalén como una ciudad que se ha vuelto corrupta y violenta, y que se ha unido con otros grupos para llevar a cabo acciones malvadas.
